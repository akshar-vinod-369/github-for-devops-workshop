The Challenge: Palindromic Sum of Consecutive Squares

In an effort to structure these terms, I propose the following challenge:

Find for all the squares s1 and the 'next' higher square s2 (s2 > s1) so that their sum is palindromic.

Let me start with some initial terms of s1 + s2 = Pal

  • 12 + 22 = 5
  • 22 + 202 = 404
  • 32 + 182 = 333
  • 42 + 142 = 212 (Note: 212 is not a palindrome, must be an error in the original list)
  • 52 + ? = ?

  • ...
